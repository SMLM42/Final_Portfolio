About Me

Hi there! My name is Steven Meiers and I am a full-stack developer using my background in tutoring to build a more user friendly experience on the web. I have recently earned a Full-Stack Development Certification from UCLA extension where I learned how to build full-stack applications from the ground up.

This course taught me both the legacy ways of code development using simple tools like HTML combined with CSS and JavaScript all the way up to modern methodologies using tools like the MERN stack (MongoDB, Express, React, Node). Known as a hard working and innovative problem problem solver who caught the errors others have missed, my aim is to make all my end products as user friendly and intuitive as possible.

Recently, I have worked on a team of five to develop the entirety of a portal for a Biotech startup that enables all of the company’s nutritionists to seamlessly and effortlessly keep track of all their clients. I am excited to leverage these skills in the future as part of a fast-paced, quality-driven team in order to build better experiences on the web.
